Easy Bridge

Timecode Protocol Bridge – User Guide Міст протоколів тайм-коду – Посібник користувача
Easy Bridge receives a timecode signal from one source (LTC audio, MTC MIDI, Art-Net, or OSC) and simultaneously re-transmits it as up to three outputs: LTC audio, MTC MIDI, and Art-Net timecode. Each output can be independently enabled, offset, and configured. The app runs as a background process and can minimise to the system tray.
Easy Bridge отримує сигнал тайм-коду з одного джерела (LTC аудіо, MTC MIDI, Art-Net або OSC) та одночасно ретранслює його до трьох виходів: LTC аудіо, MTC MIDI та Art-Net тайм-код. Кожен вихід можна незалежно вмикати, зміщувати та налаштовувати. Застосунок працює у фоновому режимі та може згортатися до системного трею.

TroubleshootingУсунення несправностей

ProblemПроблемаWhat to checkЩо перевірити
Status stays STOPPED with LTCСтатус залишається STOPPED при LTCVerify the correct input device and channel are selected. Check the Level meter — if it is empty, no audio is reaching the device. Adjust Input Gain if the signal is too weak. Confirm sample rate matches the device.Перевірте правильність вибору вхідного пристрою та каналу. Перевірте індикатор рівня — якщо він порожній, аудіо не надходить до пристрою. Відрегулюйте Input Gain, якщо сигнал занадто слабкий. Переконайтеся, що частота дискретизації відповідає пристрою.
No MIDI ports listedMIDI-порти не відображаютьсяInstall or reinstall your MIDI driver. On macOS the built-in IAC driver can be enabled in Audio MIDI Setup. On Windows use loopMIDI for virtual ports.Встановіть або перевстановіть MIDI-драйвер. На macOS вбудований IAC-драйвер можна увімкнути в Audio MIDI Setup. На Windows використовуйте loopMIDI для віртуальних портів.
Art-Net not receivedArt-Net не приймаєтьсяCheck firewall rules for UDP port 6454. Make sure the Listen IP matches the adapter connected to the Art-Net network.Перевірте правила брандмауера для UDP-порту 6454. Переконайтеся, що Listen IP відповідає адаптеру, підключеному до мережі Art-Net.
OSC not receivedOSC не приймаєтьсяVerify the Port matches the sender. Check the adapter / Listen IP setting. If sender and receiver are on the same machine, use 127.0.0.1.Перевірте відповідність порту відправнику. Перевірте налаштування адаптера/Listen IP. Якщо відправник і приймач на одній машині, використовуйте 127.0.0.1.
LTC output sounds distortedВихід LTC звучить спотвореноReduce the Output Level slider. The LTC signal should be clean; clipping causes decode failures downstream.Зменшіть повзунок рівня виходу. Сигнал LTC повинен бути чистим; перевантаження спричиняє помилки декодування.
Sync drift between outputsЗсув синхронізації між виходамиUse the Offset sliders on each output to compensate for downstream latency. Check Processing Latency in the Status Monitor window.Використовуйте повзунки Offset на кожному виході для компенсації затримки. Перевірте Processing Latency у вікні Монітора статусу.
ASIO driver failed to loadASIO-драйвер не завантаживсяEnsure you are running the 64-bit build of Easy Bridge and that a 64-bit ASIO driver is installed (e.g. ReaRoute x64 from Reaper).Переконайтеся, що запущена 64-бітна збірка Easy Bridge і що встановлений 64-бітний ASIO-драйвер (наприклад, ReaRoute x64 від Reaper).
Audio devices missing after reconnectАудіопристрої зникли після перепідключенняUse Settings → Rescan audio devices to refresh the device lists.Використовуйте Settings → Rescan audio devices для оновлення списків пристроїв.
LTC Thru produces silenceLTC Thru не передає звукEnsure the LTC Input source is selected and running (the Level meter should show a signal). Thru requires an active LTC input — it copies the same audio channel being decoded. Confirm the Out LTC switch is ON and the Thru dot is ON. Select the correct output device and channel.Переконайтеся, що джерело LTC Input вибрано та запущено (індикатор рівня повинен показувати сигнал). Thru вимагає активного LTC-входу. Переконайтеся, що перемикач Out LTC увімкнено і крапка Thru увімкнена. Виберіть правильний вихідний пристрій та канал.
MTC Output auto-switched or disabled unexpectedlyMTC Output автоматично переключився або вимкненийThe MTC loop guard activated because the selected Output port matched the Input port while MTC was the active source. Choose a different Output port, or use a virtual MIDI loopback (e.g. loopMIDI) to create a separate port.Захист від петлі MTC активовано, оскільки вибраний порт виходу збігся з портом входу при активному MTC. Виберіть інший порт виходу або використовуйте віртуальний MIDI-зворотний зв'язок (наприклад, loopMIDI) для створення окремого порту.
